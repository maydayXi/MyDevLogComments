 # 部落格留言

 本空間用於儲存並管理我的部落格留言。

 留言是使用 [Giscus](https://giscus.app/zh-TW) 建置並管理，整合 GitHub Discussions.

 ## 運作方法

 - 在部落格上留言時，留言會存在這個專案的 GitHub Discussions。
 - 需要有 **GitHub 帳號** 才可以留言。
 - 所有留言是公開的

## 部落格

歡迎瀏覽 [MyDevLog](https://maydayxi.github.io/MyDevLog/)

## 留言規範

- 請保持友善與尊重，避免不當言論
- 禁止垃圾留言或自我宣傳
